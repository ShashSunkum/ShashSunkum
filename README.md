## 🚀 Things I've built

### 🏭 WMS AI Assistant @ NCS Partners
> *Natural language querying for warehouse operations — so floor workers can ask questions instead of running reports*

Built on a hybrid RAG architecture: **FastAPI + PostgreSQL/pgvector + AstraDB + Celery + Redis**, with MCP-based agentic capabilities and a **React Native** cross-platform UI. The kind of project where getting the data pipeline architecture right actually matters.

---

### 🏷️ [DiscountBytes](https://github.com/ShashSunkum/DiscountBytes) — Senior Capstone
> *Dynamic pricing using computer vision. 92% accuracy in pilots. 18% estimated labour cost reduction.*

`React` `TypeScript` `Python` `OpenCV` `Firebase` `GKE`

What started as a Flask/Docker MVP became a cloud-native backend on **Google Kubernetes Engine** that cut API latency by 40%. Learned more about system design from this one project than anything else.

---

### ⚽ [FUT.AI](https://github.com/ShashSunkum/FUT.AI)

`Python` `React` `Flask` `Supabase` `Docker` `Kubernetes`

Four microservices. Multiple external APIs. One Bash CLI that ties it all together. Built because I wanted the data — and then it turned into an actual architecture exercise.
